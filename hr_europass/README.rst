Usage
=====

This module enables you to manage Europass's CV into Odoo.

Features:
---------
* Upload CV
  * Create it,
  * Update it
* Search on CV content
* Send a CV by Email to request modification
* Automatically upload CV in Open ERP with a CV sent to an alias email
